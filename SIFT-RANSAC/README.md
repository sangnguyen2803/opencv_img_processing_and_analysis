## Summary

These scripts demonstrate image preprocessing, feature detection (SIFT), matching, and analysis workflows used in the case studies. They were written as part of an assignment or exploration into local feature methods and image matching.

## Prerequisites

- Windows (instructions below use PowerShell)
- Python 3.8 or newer
- Recommended packages: `opencv-contrib-python`, `numpy`, `matplotlib`, `scikit-image`

Note: SIFT is part of OpenCV's contrib modules; installing `opencv-contrib-python` is recommended so SIFT is available as `cv2.SIFT_create()` (or via `xfeatures2d` in older code).

## Quick setup (PowerShell)
```powershell
# create a virtual environment
python -m venv .venv
# activate it
.\.venv\Scripts\Activate
# upgrade pip
python -m pip install --upgrade pip
# install required packages
pip install opencv-contrib-python numpy matplotlib scikit-image
```

If you prefer to use a `requirements.txt`, create one with the packages above and run `pip install -r requirements.txt`.

## How to run the scripts

From PowerShell with the virtual environment activated, run:

```powershell
python .\case_study1_p1.py
python .\case_study1_p2.py
python .\case_study2.py
python .\case_study3.py
```

Notes:
- The scripts in this repo may read images from `images/`.
